Useful Commands

/tpa - Request teleport

/tpaccept - Accept teleport requests

/tpdeny - Deny teleport requests

/rtp - Random Teleport

/home - Teleport home

/sethome - Set your home

/delhome - Delete a home

/spawn - Return to spawn

/shop - Open shop

/auction - Open auction house

/warp - Warp to locations

/bal - Check your balance

/pay - Pay other players

/msg - Private message players

/afk - Toggle AFK mode

/ignore - Ignore players
