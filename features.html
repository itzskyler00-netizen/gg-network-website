How To Join Discord

Features

⚡ 24/7 Online

The server stays online all day and night so you can join anytime and play with friends.


🌎 Java + Bedrock Support

Java and Bedrock players can play together on the same server.


🏪 Economy & Shops

Earn money, buy items, sell resources and build your own economy.


💰 Auction House

Trade items with other players using the Auction House system.


🎙 Voice Chat

Talk with nearby players using Simple Voice Chat.


🏠 Better Homes

Set homes and travel quickly across the server.


🎲 Random Teleport

Use RTP to explore new locations and start your adventure.


⚔ PvP & Arenas

Challenge players, fight in arenas and prove your skills.


🛡 Protection Plugins

Advanced protection systems help keep the server safe and stable.
